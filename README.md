# Vølven PHP Debt Skill

Vølven sees the PHP code that will hurt later.

This skill reviews PHP for future cost: weak array contracts, unsafe `mixed` boundaries, magic workflow strings, hidden helper dependencies, misleading PHPDoc, and shallow tests.

It does not replace Pint, PHPStan, Psalm, Rector, or project standards. Those tools already have jobs. This skill looks for the debt pressure that survives after those tools go green.
